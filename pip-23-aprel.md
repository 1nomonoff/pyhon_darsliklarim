
pip- paket manager

Package install qilish : 

1 : pajetarni web saytdan korish uchun pypi.org 

2 : qidiruvga package nomini yozish kerak 

3 : pip install paket-nomi -> ornatish uchun

4 : pip list -> paketlar listini korish uchun

5 : pip uninstall package_nomi -> ochirish uchun : agar y/n soramasin dsen oxirigia -y qoyb ket !

5.1 : Y/N soridi ->tastiqlash yoki rad etish : 
5.2 : Y -> tastiqlash package ochadi 
5.3 : N -> rad etish package ochmidi
